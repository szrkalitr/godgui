## 🚀 Kullanım

1. Termux’ta gerekli paketleri kur:
    ```bash
    git clone https://github.com/szrkalitr/godgui
    pkg install x11-repo
    ```

2. Scripti çalıştır:
    ```bash
    cd godgui
    mv gui.sh ~/
    chmod +x szrgui.sh
    ./szrgui.sh
    cd ~/
    chmod +x gui.sh
    ./gui.sh
    ```
